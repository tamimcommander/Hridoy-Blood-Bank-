# 🩸 Hridoy Blood Bank

A simple Blood Bank Management System built with HTML, CSS, JavaScript and Firebase.

---

## 🚀 Features

- 🩸 Donor registration system  
- 📞 Direct call button  
- 🔍 Blood group search filter  
- 📊 Real-time dashboard (Firebase)  
- 🌙 Dark mode support  
- 👨‍💼 Admin panel with support info  
- 💬 Messenger group link  

---

## 🛠 Tech Used

- HTML5  
- CSS3  
- JavaScript  
- Firebase Realtime Database  

---

## 📂 How to Use

1. Download or clone this repository  
2. Open `index.html` file in browser  
3. Add donor information  
4. Data will save in Firebase automatically
